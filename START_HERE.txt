Q: Which file do I edit?
A: Usually just app.py if you want to change the web interface.
   The model training is in model_building.ipynb (don't need to change).

Q: Where do I get the data?
A: Data is downloaded automatically in the notebook from public sources.
   You don't need to download anything manually.

Q: How do I run this locally?
A: 1) pip install -r requirements.txt
   2) streamlit run app.py
   3) Open browser to http://localhost:8501

Q: How do I train the model?
A: Open model_building.ipynb in Google Colab and click "Run all"
   Wait ~3 minutes. Download the .pkl file when done.

Q: What if the model file is too big?
A: GitHub has a 100MB limit. If .pkl is >100MB, use Git LFS
   (most likely won't be an issue - should be ~15-20MB)

Q: Can I change the 6 features?
A: Yes! Edit model_building.ipynb cell that says "selected_features"
   Then retrain and redeploy.

Q: Do I need to change app.py?
A: Only if you change features in the notebook.
   The form automatically matches the features used in training.

Q: What's the Render start command?
A: streamlit run app.py --server.port=10000
   This is already in SETUP_GUIDE.md

Q: How do I know if deployment worked?
A: Visit your live URL. If the form loads, it worked!
